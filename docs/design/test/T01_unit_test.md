# T01 単体テスト仕様書

作成日: 2026-06-07

---

## 1. 対象・方針

- テストフレームワーク: Vitest + @testing-library/react
- 対象: ブラウザ環境・Supabase 不要で検証できる純粋関数・コンポーネント
- 対象外: Supabase クライアントを呼び出す loader・非同期ハンドラ（T02 で対象）
