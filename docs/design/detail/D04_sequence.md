## 2. 参加者のルーム参加〜投票

```
参加者（ブラウザ）        フロントエンド（P03）          DB（rooms / votes）
    │                        │                              │
    │ 参加者URLを開く         │                              │
    │ /room/:participantToken │                              │
    │───────────────────────>│                              │
    │                        │ [loader] SELECT rooms        │
    │                        │ WHERE participant_token = :token                   │
    │                        │─────────────────────────────>│
    │                        │<─────────────────────────────│
    │                        │                              │
    │                        │ expires_at < NOW() ?         │
    │                        │ → YES: P07（期限切れ）表示   │

    │                        │ localStorage.getItem(        │
    │                        │   `voted_${roomId}`) === 'true' ?               │

    │<───────────────────────│                              │
    │                        │                              │

    │───────────────────────>│                              │
    │                        │ バリデーション               │
    │                        │                              │

    │                        │ { room_id, choice }          │
    │                        │─────────────────────────────>│
    │                        │<─────────────────────────────│ 成功

    │<───────────────────────│                              │
```

## 4. ルームの自動期限処理

### 4-1. フロントエンドでの期限切れ検知

```
参加者（ブラウザ）        フロントエンド（P03 / P05 / P06）
    │                        │
    │ ページアクセス          │
    │───────────────────────>│
    │                        │ rooms.expires_at を取得
    │                        │
    │                        │ expires_at < NOW() ?
    │                        │ → YES: P07（期限切れ）インライン表示
    │                        │ → NO: 通常表示継続
    │<───────────────────────│
```

P05・P06 は表示中に expires_at を監視しない（ページをリロードした時点で判定）。なお、期限切れ後は RLS の INSERT ポリシーにより DB レベルでも投票が拒否される。

### 4-2. データ自動削除（Cron）

```
pg_cron / Supabase Scheduler    Edge Function（delete-expired-rooms）    DB（rooms）
    │                                       │                               │
    │ 毎日 02:00 UTC にトリガー             │                               │
    │──────────────────────────────────────>│                               │
    │                                       │ DELETE FROM rooms             │
    │                                       │ WHERE expires_at <            │
    │                                       │   NOW() - INTERVAL '30 days'  │
    │                                       │──────────────────────────────>│
    │                                       │                               │ CASCADE削除
    │                                       │                               │ （votes_return_time）
    │                                       │                               │ （votes_afterparty）
    │                                       │<──────────────────────────────│
    │                                       │ 完了ログ出力                  │
```
